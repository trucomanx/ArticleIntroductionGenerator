# article-introduction-generator

Program to generate a prose-style introductory text from a structured JSON file.

## Install from PYPI

The homepage in pipy is https://pypi.org/project/article_introduction_generator/

```bash
pip install --upgrade article_introduction_generator
```

Using:

```bash
article-introduction-generator
```

## Install from source
Installing `article-introduction-generator` program

```bash
git clone https://github.com/trucomanx/ArticleIntroductionGenerator.git
cd ArticleIntroductionGenerator
pip install -r requirements.txt
cd src
python -m build
pip install dist/article_introduction_generator-*.tar.gz
```
Using:

```bash
article-introduction-generator
```

## Uninstall

```bash
pip uninstall article_introduction_generator
```
